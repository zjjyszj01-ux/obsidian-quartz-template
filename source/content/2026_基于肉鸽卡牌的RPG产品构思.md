## 5.3 战斗流程

1. **抽牌阶段**：每回合获得初始能量与手牌。
	<br>
	
2. **部署阶段**：消耗费用打出四类卡牌——
	①单位卡：放在下三层的任意格位，通常按前排承伤、后排输出/辅助的方式布阵；
	
	②法术卡：即时生效，造成伤害/Buff/Debuff等效果；
	
	③装备卡：给单位穿戴，提供属性与能力；
	
	④房间卡：安装到某一层，提供全层属性加成或环境规则。
	*<span style="color:#E63946">\*以消灭棘手怪物或加速成长为目标的运营环节，智力取胜的爽感在此环节释放。</span>*
	<br>
	
3. **结算阶段**：该回合敌人行动、双方触发技能关键词（燃烧、英勇、再生、充能、转移、月相等），前排先交火。
	*<span style="color:#E63946">\*单位极速成长、疯狂斩杀怪物的爽感在此环节释放。</span>*
	<br>
	
4. **波次推进**：每场战斗 6-8 回合，回合结束时，每层房间剩余怪物上升1层，最下层刷出新的怪物波次；最后一波为 Boss ，数值夸张，且 BOSS 自最下层开始触发 **死斗** —— 直至 BOSS 死亡或该层所有己方单位被消灭才结束本层战斗。
	*<span style="color:#E63946">\*单局思路为尽快成长，通过前几回合的 单位协同 + 装备 + 房间 + 法术 叠出大数值消灭最终BOSS。</span>*

## 6.4 增加“侵蚀”机制

- 增加 ”**侵蚀**” 机制，将 ”**微恐**“ 与  “**色气**” 整合进 “**玩法**”，体验接近《卡厄思》的 “**崩溃**”机制，并优化2个关键体验。
	 1. “**侵蚀**”机制 ：独立于卡费的一套资源循环。**常规战斗过程积攒灵视** => **使用灵视释放必杀** => **角色侵蚀度增加** => **高侵蚀度下几率进入侵蚀状态** （风险博弈1）=> **侵蚀后角色宕机需要打出侵蚀牌恢复** => **影响出牌策略** => **恢复时瞬间获得BUFF/卡费等大量资源**（风险博弈2）；
	![[Pasted image 20260414223650.webp|center]]

	 2. **策略维度**：
		 - 基础策略：攒灵视、放必杀、控侵蚀度；
		 - 延申策略【1】：在肉鸽这样的长单局中，灵视、侵蚀度、侵蚀状态全局延续，玩家可以根据后续战斗压力选择 **是否提前引爆侵蚀状态**，避免后续高压战斗翻车；
		 - 延申策略【2】：若角色已进入侵蚀状态，且退出侵蚀状态的进度乐观，则可**有意拖延退出进度至BOSS战，充分利用退出侵蚀状态瞬间的BUFF和额外费用**；
	 3. **灵视** ：公共战斗资源。 随战斗中的常规行为积攒，使用角色必杀时消耗；
	 4. **侵蚀度**：每个角色独立计数的百分比数值。随回合数、受伤、使用必杀（消耗灵视）增加，受到有效治疗时减少，100% 时必定进入侵蚀状态；
	 5. **侵蚀状态**：
		 - **逻辑层**：
			 - ①角色无法正常行动。需要打出侵蚀卡牌恢复，影响出牌决策，且打出时角色会触发随机行为（如攻击身边队友）；
			 - ②退出侵蚀状态时，获得额外BUFF和费用，瞬间减轻战场压力，是对用户”坚持到了角色恢复正常状态“的强烈正反馈；
		 - **表现层**：角色形象变得崩坏、高露出，添加"Ahego"元素；
		 ![[Pasted image 20260413181809.webp]]
		<br>
	 6. **改良后的循环也一并解决了《卡厄思梦境》的两个体验问题**：
		 - 《卡厄思》的崩溃状态是被动的、不可控，后期数值高了难触发崩溃状态，欣赏不到崩溃立绘，**作为战中策略的一环、以及“色气”亮点大幅削弱**；*<span style="color:#E63946">改良后的循环，可主动触发、可有意图的指定对象触发</span>*。
		 - 《卡厄思》是 3 套资源循环：压力值=>崩溃状态，意识点数=>伙伴技，费用=>出牌，**信息负担较高，且意识点数的存在感不强**；*<span style="color:#E63946">将前 2 条链路合并为 1 条，灵视=>侵蚀度=>侵蚀状态</span>*；
<br>

## 6.5 调整战斗节奏
 
- 原作单场战斗时长过长（8-12分钟），不匹配网游用户习惯，需要适度轻量化：
	1. 肉鸽局内，提高属性成长后的保留幅度（原作一局肉鸽内的几乎每场战斗都需要重新积攒），从而减少单场战斗的成长运营时间（减少小怪波次）：
		- 局内最终数值 = 存档继承数值 + 局内培养增量
	2. 战斗难度分级：
		- 【少量】BOSS战：5-6波怪，6分钟；
		- 【中等】精英战：4波怪，4分钟；
		- 【大量】小怪战：3波怪，3分钟；
	3. 战场削减至3层，2层单位层+1层基地所在，单位层最多分别部署3名角色，适当减轻策略负担和决策压力。（下方给出布局示意）

## 6.10 完善玩法框架
### 6.10.1 核心思路
1. 以PVE为主，在保持玩法周期性更新的同时，借鉴《卡厄思梦境》的**赛季活动框架**。
2. 加入 ”异步构筑对抗“ 玩法。
3. 加入符合客群属性的 XP 玩法。
4. 多角色需求按2队（PVE终局）、无上限（PVP玩法）的方式引导玩家养成，保底需求 10 人以上；
5. 更强调内容和环境的变化，而非单纯的数值压力；
<br>

### 6.10.2 PVE模块
- 以 **局内肉鸽搜打撤** + **局外常规数值校验** 作为基础框架。
#### ① 常驻肉鸽（局内）
- 参考《卡厄思梦境》，划分多个章节，每个章节的事件、BOSS、装备掉落不同，承担引导、基础构筑刷取的职责.
	1. 设计要点：
		- 构筑丰富：单卡强化方向多、装备/中立卡丰富、移除/复制等行为有时机变化；
		- 事件厚度：单事件的选择变化多、多事件之间有上下文关联，有涌现惊喜；
		- 风险博弈：即战力(装备/中立卡) vs 经济运营（金币）、局内构筑强度 vs 想带出的理想存档；
		- 奖励循环：通关奖励、探索进度奖励、成就奖励、局外养成等，一次下本超多收获；
	2. 基础流程：
		- 类《杀戮尖塔》，共2面，每面多路线选择，BOSS镇守关底，队伍状态延续，击败第2面BOSS后可通关撤离；
		![[Pasted image 20260415140129.webp|center]]
	3. 搜打撤元素：
		- 【搜】：大量随机事件和盲盒奖励，搜刮卡牌神闪、高品质装备、中立卡等高价值奖励；
		![[Pasted image 20260415164812.webp|center]]
		- 【打】：应对不同的怪物组合，击败关底BOSS，验证局内构筑；
		![[Pasted image 20260415164836.webp|center]]
		- 【撤】：主动或通关撤离时，除材料奖励外，还可以带出卡牌构筑和装备；
		![[Pasted image 20260415164851.webp|center]]
	4. 主题区分：按主线章节解锁不同主题，在怪物种类、随机事件、卡牌神闪、装备掉落、BOSS设计上有明显差异，且场景氛围、事件演绎充分，叙事代入感强；
		![[Pasted image 20260415165005.webp|center]]
	5. 探索进度：每个主题肉鸽的掉落内容（中立卡牌/独特装备）按探索进度解锁，平衡内容消耗、用户耐心和活跃诉求；
		![[Pasted image 20260415165033.webp|center]]
	6. 外围养成：肉鸽代币可提高增益或解锁额外功能，提升内容丰富度、减轻重复探索的数值压力；
		![[Pasted image 20260415174844.webp|center]]

### 6.10.3 PVP模块

#### ① 肉鸽竞技
- 异步构筑对抗玩法，更纯粹的斗蛐蛐体验，框架参考《大巴扎》，具体规则迭代适配；
- 核心设计：
	1. 胜负规则：初始3点生命，对战事件失败扣除1点生命，取得 **5胜或扣除全部生命** 后结束，按照胜场获得积分，并进行排名。
	2. 单局流程：单局总计 7 轮事件，每轮由 6 个回合组成，前 5 回合的事件服务于构筑，最后 1 回合固定为 **对战事件** ，对战事件的胜负影响胜场和生命扣除；
	![[Pasted image 20260417110227.webp|center]]
	
	3. 队伍人数：初始从自身拥有的英雄中选择3位，局内可以继续征召公共/自身拥有的英雄，最多3名；
	4. 匹配规则：对战事件中，与同分段同轮数的玩家镜像对战；
	5. 战斗模型变化：
		- 简化为单层对战（类PVE模式下的BOSS死斗环节），无基地，去除第2-3层；
		- 最多允许出战4人，战斗胜利目标为全灭敌人；
		- 新增速度值设定，出牌阶段按单位的速度值确定出手顺序，出手时仅能使用自身卡牌（平衡先后手）；
		- 速度值由局内加点获得，可通过事件和商店花金币提升；
		- 每个大回合初始可用9点费用，每一方所有角色之间共用（即我方A角色先用掉5点，剩下的B、C、D角色一共只能再用4点）；
		- 敌方出牌阶段 AI 托管；
		- 大回合结束时，进入拼刀阶段，同样按出手速度决定拼刀顺序；
	6. 存档规则：局外存档不可用，进入肉鸽竞技时所有角色的卡组初始化；
	7. 构筑内容：
		- 装备；
		- 中立卡；
		- 房间卡（对战时敌我双方的房间卡同时存在）；
		- 卡牌操作（移除、复制、镶嵌）；
		- 角色加点（百分比提升基础属性，或提升速度值）；
	8. 初始商店：初始提供300金币和1次商店事件（可进行卡牌操作、角色加点、房间卡购买、装备购买等）
	9. 段位重置：每 3周/6周 重置段位并发放排名奖励（段位+名次）；
	10. 更新内容：公共英雄轮转，新的装备/房间卡加入，新的赛季怪物等；

#### ③赛季BOSS（终局）
- 体验参考 《本体》 - 泰坦战  + 《卡厄思》-大龟裂，划分为两个阶段，每个阶段需求1队挑战：
	1. 【阶段1】：
		- 1只巨型BOSS，巨型BOSS的部位横跨 3 层，血量共享，攻击力分层；
		- BOSS不触发死战规则；
		- 消灭巨型BOSS视为胜利；
		- 胜利时将剩余回合数转移给【阶段II】，最多5回合；
	2. 【阶段2】：
		- 1只巨型BOSS + 1只小型BOSS，巨型 BOSS 部位横跨 3 层，小型 BOSS 在1、2层之间不断移动（任意BOSS存活时，另一只BOSS生命最小为1，照常行动）；
		- BOSS不触发死战规则；
		- 消灭巨型BOSS和小型BOSS视为胜利；
	3. 我方基地被摧毁，或【阶段2】胜利，或回合数耗尽时结算；
	4. 结算时根据两个阶段的总积分发放周常和成就奖励，并进行排名，赛季结算时发放段位和排名奖励；
	5. 积分来源：伤害量（衰减）、治疗量（衰减）、护甲量（衰减）、单次伤害量等；
- 环境特点：纯BOSS挑战，开局高压，跨楼层的BOSS血厚，有小体型BOSS在前方巡航，生存、DPS、牌组运营（有卡手机制） 3 重检验；
- 考验能力：同时考验生存、DPS、牌组运营；
- 更新周期：在赛季中、下两段更新；

### 6.10.4 XP 玩法
- 加入面向用户 XP 的情感互动玩法，强调用户与角色情感的深化，提供情绪价值：
	- 借助 ”微恐“ 题材和 ”侵蚀状态“ 的特点，设计 **沟通引导 => 揭露创伤 => 角色进入侵蚀状态 => 被主角救赎 => 对主角表达好感** 的 玩法互动循环 ，建立起用户对角色的深度情感链接；
	- 交互过程以近距离观测侵蚀创伤和选择治疗手段作为包装，放大观察角色的侵蚀立绘，选择创伤的外部治疗手段（触碰/工具治疗等），治疗时角色给与高亢反馈，并结算治疗效果，多轮后完成 ”创伤“ 治疗；
	- 情绪价值输出：
		- 角色的侵蚀状态立绘观察；
		- 治疗时的高亢反馈；
		- ML向语音；
		- 好感度提升后的其他奖励解锁；
