# **1**、【前言】 

## 1.1 产品成绩 

明日方舟上线首月，基本维持在畅销榜前5，预估全渠道流水6亿，制作团队则是 17年初创的鹰角，是19年截至目前最大的一匹黑马。方舟的成绩单在一定程度上教育了开发者，用户的口味和偏好，在不断发生偏移，对于差异化的玩法体验的诉求，也在不断加深。

## 1.2 产品类型 

类型上，明日方舟归属为SRPG（策略角色扮演），核心玩法则脱胎于TD（塔防游戏）。在保留“部署防御塔与消灭敌人”的核心规则下，明日方舟引入了新颖的废土世界观，塑造了各具特点的角色，并且在战斗策略层面做了极大的丰富，拓展出玩家对各类干员（防御塔）的需求，支撑起了游戏的付费。这些调整也使得它显著区别于传统 TD 的轻量化及付费能力。

# **2**、【战斗策略分析】 

本文中我将针对方舟的战斗策略做一些分析，希望能够将这个游戏运作的核心驱动力呈现出来。

我们以关卡体验作为发掘方舟战斗策略的研究对象，以提问的方式代入到设计者的角度来剖析怎样设计出策略丰富的战斗体验：

- **Q1：最基本的关卡流程是怎样的？是否易于上手？**

- **Q2：关卡战斗中有哪些核心机制？分别有什么作用?**  

- **Q3：关卡由哪些基本元素组合？每个元素的可拓展性和丰富度如何？**

- **Q4：关卡内的节奏变化如何？是否能够产生心流？**

## 2.1 关卡流程 

- Q1：最基本的关卡流程是怎样的？是否易于上手？

- 明日方舟的关卡基本流程可以概括为“在特定区域部署干员阻挡并消灭所有来袭的敌人”，与传统 TD 的核心规则一致。在初期的教学关卡中，会通过引导玩家放置具备进攻和阻挡能力的干员，消灭敌人完成关卡，反复强调这一认知，容易上手。

## 2.2 核心机制 

- **Q2：关卡战斗中有哪些核心机制？分别有什么作用?**

- 与传统TD相比，方舟增加或改良出以下设定。

### 2.2.1 费用设定 

![[Pasted image 20260427145830.webp|center]]
- 【费用】：关卡中部署干员、召唤物等需要消耗费用。每个关卡的初始费用及费用回复方式存在差异。

- 实现效果：
	1. 增加局内经营策略。“费用”可以通过自然累计、击杀回复、技能产生等多种方式获得，又是干员投放的必要条件，随着敌人的强度不断增加，要求防线的部署速度越来越快，防线越来越完整，从而使得玩家不断思考①如何尽快获得更多费用？②如何部署干员发挥费用的最大利用率？
	2. 借助于费用，诞生了一类特点鲜明的职业——先锋,其独有的“击杀敌人回复费用”与“使用技能回复费用”特性，可以帮助玩家获得额外的费用，使得防线更快成型。
	3. 不同的关卡，利用费用的初始值、来源的设定不同，能够有效避免开局时千篇一律的重复体验。

### 2.2.2 阻挡 

- 【阻挡】：地面干员可以阻挡若干个经过的敌人，发生阻挡后，干员与怪物进行攻防对抗。 

- 实现效果：
	1. 使得游戏中能够采取阻挡-击杀的部署策略。 
	2. 强调了阻挡数量、生命、防御属性的价值，并诞生了三类特点鲜明的职业——同时承担阻挡/击杀任务的近卫干员、以阻挡多个敌人为特色的肉盾重装职业、以治疗伤势为主的医疗职业。 

### 2.2.3 干员朝向与攻击范围 

- 【干员朝向】：人物面向的方向，会作为干员的主要攻击方向，通常情况下，人物无法攻击身后的敌人。 
![[Pasted image 20260427145918.webp|center]]

- 【攻击范围】：指干员面朝方向的攻击区域大小，与自身的职业以及细分定位有关，远程干员的攻击范围一般较大，近战干员的攻击范围一般较小。 
![[Pasted image 20260427150020.webp|center]] 

- 实现效果：攻击范围受干员朝向影响这一规则，引出了防守方向的部署策略，使得玩家思考，如何让干员的攻击范围在防线前密集交叉，提高防线质量。 

### 2.2.4 撤退/再部署 

- 【撤退】： 
	1. 主动撤退：玩家主动将已部署的干员撤退，作为主动采取的战术意图，会返还部分费用。 
	2. 被动撤退：干员被怪物击杀时发生，作为惩罚，不会返还费用。 

- 【再部署】：已经撤退的干员，可以再次进行部署，但需要等待较长的冷却时间，并支付随再部署次数不断增长的高额费用。 

- 实现效果：撤退与再部署的机制，使得玩家可以灵活调整防线，但同时也设立了长冷却时间和更高的部署费用，提高玩家的决策成本以提醒玩家谨慎决策。 

## 2.3 交互元素 

- Q3：关卡由哪些基本元素组合？每个元素的可拓展性和丰富度如何？

- 如图所示，方舟的关卡交互元素共分为 4 类，即地形、放置物、怪物、环境， 4 种元素各自的分类繁多，组合搭配产生了极为丰富的关卡体验。 
![[Pasted image 20260427150045.webp|center]]

### 2.2.1 地形 
![[Pasted image 20260427150212.webp|center]]
- 概念：
	1. 地形是组成关卡的基本单位，每个关卡均由 M x N 块地形格子组成（如上图） 
	2. 不同地形的效用不同，方舟中地形的分类大致如下： 

	![[Pasted image 20260427150409.webp|center]] 
	- 地形最基本的两个属性是“**敌人是否可行走**”以及“**是否可摆放干员**”，连续的可行走地形组合出了敌人的行进路线，在读懂行进路线后，玩家需要借助可摆放干员的地形，思考狙击防线的布置，这是在刚进入关卡时，玩家最优先需要思考的部署策略。 
	- 特殊地形带有“属性影响”或“特殊效果”，在单一关卡中较少配备以突出战略意义（例如陷阱类的地形，会对经过的敌人产生即死的效果，该地形搭配队伍中“使敌人产生位移”效果的干员，可以极大缓解正面对抗敌人的压力） 

- 拓展性：作为基本元素，地形的可拓展性很强，无论是从自身的独特效果出发还是通过地形间的搭配组合，都能够带来独特的关卡体验。 

### 2.2.2 放置物 

![[Pasted image 20260427150424.webp|center]]

- 概念：放置物指玩家可以手动部署的单位，分为干员、召唤物、障碍物三类，每次部署占据一个可放置的地形，关卡中通过右下角的浏览，可以看到放置物的职业（类型）、消耗费用、状态等信息。
	1. 干员：干员是方舟中的核心收集和养成对象，本文只简单罗列出干员的基本要素，引出职业定位和战略属性这两个核心特点：
![[Pasted image 20260427150438.webp|center]]


	- 干员拥有清晰的职业定位，在坦克、输出、治疗的铁三角以外，还加入了强化/削弱定位的辅助、拥有特殊能力的特种以及能够产生额外费用的先锋等，扩张了搭配策略和养成维度。
	- 在同一职业大类下，通过技能、天赋、战略属性等可以进一步区分每个干员的细分定位，确保对每个干员的追求与培养价值。（实际上存在重复定位的高星与低星干员，但游戏通过降低培养成本的设定，确保了低星干员有过度使用的价值）
<br>


2. 召唤物：召唤物是部分干员携带的额外放置物，将相应干员编入队伍后，即可在关卡中放置使用，召唤物常作为各种定位的干员的临时补充，可起到应急救场的作用
	![[Pasted image 20260427150456.webp|center]]
	<br>

3. 障碍物：障碍物实际上是召唤物中特殊的一类，由系统自动配置到关卡当中，玩家可以使用有限的障碍物，改变关卡地形，创造有利于防线部署的怪物路线：
	![[Pasted image 20260427150514.webp|center]] 

- 拓展性：
	1. 针对干员，干员养成是用户黏度的保证，干员抽取是游戏的核心付费点，随着剧情推进，新角色的出现以及干员本身丰富的维度，其拓展必要且简便。
	2. 召唤物作为干员的附属，其种类必然会随着干员的拓展而拓展。
	3. 障碍物依附于关卡自身，属于关卡特点，势必会随着关卡的拓展而拓展。


### 2.2.3 怪物 

![[Pasted image 20260427150530.webp|center]]

- 概念：怪物是玩家在关卡中的对抗目标，主要通过数值强度、技能和行为AI三部分区分类型，怪物类型的不同，会显著影响玩家对关卡的应对策略。
	1. 数值强度：怪物的属性类型主要包含以下几项内容，根据怪物自身定位及形象可以揉捏出各种强度的模板，值得一提的是TD中关卡的起点与终点距离固定，因此怪物的移动速度快慢也会显著影响怪物的强度：
	![[Pasted image 20260427150550.webp|center]]

	2. 技能：除常规的伤害、控制、治疗效果外，还包含被动隐身、无视阻挡、改变地形等针对关卡核心机制的特殊效果，在此不一一列举。
	3. 行为AI：除常规的行走、反击的AI基础上，还加入了大量的条件状态AI，如低血狂暴、死亡自爆、优先选取目标等等，效果繁多，不一一列举。
<br>

- 拓展性：方舟中的怪物，在设定的丰富度上，不亚于一些MMO产品，目前BOSS 关卡中，每个精英怪、BOSS的技能效果、行为AI都相当丰满，可以想见后续关卡中也必然会有更多不同强度定位和行为丰富的怪物出现。

### 2.2.4 环境 

![[Pasted image 20260427150607.webp|center]]

- 概念：环境指关卡中针对我方或敌人生效的机制，通常以 buff 或 debuff 的形式存在，针对全图生效，威力较强。 

- 拓展性：方舟中截至目前，拥有环境效果的关卡还不多见，但作为一类独立影响关卡难易和独特性体验的因素，后续必然会有更多拓展效果。 

## 2.4 关卡节奏设计 

- Q4：关卡内的节奏变化如何？是否能够产生心流？

- 在非数值碾压的前提下，方舟在关卡体验上基本遵循 2 种设计模式： 

	1. 前易后难：常为主线关卡，难度循序渐进，拥有特性的敌人种类较少，玩家在不断部署干员和准备迎接高强度敌人的预期心理下，平稳进入到关卡的高难部分，更多考验玩家基本的防线成型能力。 

	2. 持续高难：常为挑战型关卡，初始敌人的强度较高，费用紧张，使得玩家需要谨慎考虑每一个干员的摆放，随着关卡进程不断推进，不同特性的敌人不断涌现，已成型的防线经常需要根据特定敌人的出现进行调整，更多考验玩家进阶的防线调整能力。 

- 费用的设定，引出了关卡内的经营策略，经营的过程中玩家不断获取和消耗费用，完善防线，随着敌人波次强度的提升，防线的强度与敌人的强度在部署过程中交替上升，短时间内持续的思考、对抗与验证，容易将玩家引入心流。 

![[Pasted image 20260427150627.webp|center]]

## 2.5 总结 

- 方舟在保留传统塔防“部署防御塔消灭所有敌人”的核心规则下，拓展出费用、阻挡、攻击朝向、再部署等核心机制，又确立了地形、放置物、怪物、环境 4 个元素的关卡搭配框架，使玩家可以在短时间的单局战斗中，体验到丰富的经营、对抗乐趣。 

- 关卡内玩家不断获取和消耗费用，完善防线，随着敌人波次强度的提升，防线的强度与敌人的强度在部署过程中交替上升，短时间内持续的思考、对抗与验证，能够很好地将玩家引入心流。 

# **3**、【问题与建议】 

## 3.1 问题一：关卡难度不足与干员强度不断提升之间的错位 

-  问题解析：当推荐平均等级为精英 1 -70 级的副本可以由一堆平均等级为精英 1-20 级的干员挑战成功时，除去当下的成就感和爽快感，随之而来的问题则是干员后续漫长养成的意义，这个问题实际上也反应出方舟的策略比重超过数值比重的设计结果（无论是在设计者的预期中还是预期外，目前的客观事实如此）。 

- 建议：加快高强度副本的推出或设计具备成长性的副本，确保有持续的内容可供玩家消耗。

## 3.2 问题二：难度可视化不足 

![[Pasted image 20260427150644.webp|center]]

![[Pasted image 20260427150658.webp|center]]

- 问题解析：以贴吧、NGA、TAPTAP等玩家论坛的互动而言，存在许多方舟玩家埋头肝养成，即使强度达标，主线关卡和高难挑战的进度却十分落后的情况，原因基本在于 1-2次的关卡尝试失败后，短时间内鼓不起勇气继续尝试，侧面反映出方舟的关卡试错成本过高，分析原因：
	1. 正式进入关卡前，一些核心信息不够透明（如初始费用），玩家无法针对性搭配干员，使得进入关卡后无人可用或陷入费用的低效使用，很快崩盘。
	2. 高强度敌人的行为模式指示不足，玩家在不了解关键敌人的情况下，部署方案存在问题，调整受限（费用、再部署时间），导致关卡失败。

- 建议：
	1. 关卡外：挑战前可查阅更多关卡信息（如初始费用、怪物种类）。
	2. 关卡内：强化BOSS的行为指示（如预期停留地点和停留时间）。

## 3.3 问题三：交互不足或门槛过高 

- 问题解析： 
	1. 添加好友门槛过高：游戏中，添加好友后，可以使用好友的支援干员协助自己通关，但添加好友需要输入确切的好友名称，且不提供好友推荐，使得该系统的门槛较高。
	2. 关卡没有合作模式：截至目前，游戏中的关卡均以单独挑战的形式存在，缺乏玩家之间的互动。

- 建议： 
	1. 增加好友推荐功能。
	2. 增加合作关卡，限定每位玩家可以选取若干位干员，组成队伍进行共同部署和挑战。

## 3.4 问题四：干员满潜追求动力不足 

![[Pasted image 20260427150711.webp|center]] 

- 问题解析：从论坛及贴吧的讨论来看，当前玩家抽取到重复 6 星干员时，带来的挫败感很强，分析原因在于抽取 6 星干员代价较高，但作为重复干员回收方式的潜能培养，没有发挥实际价值，下图以一个 6 星干员为例，来解读潜能提升效果的实际价值： 
![[Pasted image 20260427150722.webp|center]] 

- 综上所得，方舟中角色满潜的提升价值低。从正面来说，能够缩减大 R 与平民的差距，但游戏中缺乏 PK 和竞争性质的玩法，因此显得没有意义；反面来说，对于付费能力和意愿强的玩家并不友好 

-  建议：提升潜力培养的数值提升效果，或增加高潜力干员的其他特性（如解锁第 3 天赋、攻击范围加大等），以此提高游戏付费。
